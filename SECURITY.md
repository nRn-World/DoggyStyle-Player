# Security Policy

## Supported Versions

We currently provide security updates for the following versions of Doggy Player:

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Doggy Player, please send an e-mail to **bynrnworld@gmail.com**. 

All security vulnerabilities will be promptly addressed. We ask that you do not report security vulnerabilities via public GitHub issues; please use the email address above instead.

### What to expect
* You will receive an acknowledgment of your report within 48 hours.
* We will keep you updated on the progress of the fix.
* Once fixed, we will release a new version and credit you for the discovery (if desired).
